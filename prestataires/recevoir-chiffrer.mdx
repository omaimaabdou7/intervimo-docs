## Objectif

Cette page explique comment traiter un mandat reçu dans Intervimo en tant que prestataire.

Le but est de consulter la demande, vérifier si vous pouvez intervenir, puis transmettre un devis au gestionnaire.

## Quand utiliser cette page ?

Utilisez cette page quand une nouvelle demande ou un mandat apparaît dans votre espace prestataire.

Elle est utile pour :

- consulter les informations de l’intervention ;
- vérifier le problème signalé ;
- accepter ou refuser la demande si l’action est disponible ;
- préparer un devis ;
- envoyer le devis au gestionnaire.

## Avant de commencer

Avant de répondre, vérifiez les informations disponibles :

- le type de problème ;
- le lieu de l’intervention ;
- le niveau d’urgence ;
- les informations transmises par le gestionnaire ;
- les notes ou photos disponibles, si elles sont visibles.

Si une information importante manque, gardez-la en tête avant de vous engager ou indiquez-le dans vos notes si l’interface le permet.

## Étapes

<Steps>
  <Step title="Ouvrir les mandats reçus">
    Depuis votre espace prestataire, ouvrez la partie dédiée aux demandes ou aux mandats.

    Repérez la demande à traiter. Elle peut être affichée avec son statut, son secteur ou sa priorité.
  </Step>

  <Step title="Consulter le détail du mandat">
    Ouvrez le mandat pour lire les informations disponibles.

    Vérifiez le titre, la description, le lieu de l’intervention et les éléments ajoutés par le gestionnaire.

    <img
      src="/images/prestataire-reception-mandat.png"
      alt="Mandat reçu dans l’espace prestataire avec les actions accepter ou refuser"
    />
  </Step>

  <Step title="Vérifier si vous pouvez intervenir">
    Vérifiez que la demande correspond à votre activité, à votre zone d’intervention et à vos disponibilités.

    Si certains éléments ne sont pas clairs, notez les points à confirmer avant de vous engager.
  </Step>

  <Step title="Accepter ou refuser la demande">
    Si l’interface le permet, répondez à la demande.

    Acceptez si vous souhaitez poursuivre. Refusez si vous ne pouvez pas intervenir.

    En cas de refus, ajoutez une note courte si un champ est prévu.
  </Step>

  <Step title="Préparer le devis">
    Préparez votre chiffrage avec les informations connues.

    Indiquez le montant si vous pouvez déjà le confirmer. Ajoutez une note pour expliquer ce qui est inclus ou ce qui reste à vérifier.
  </Step>

  <Step title="Soumettre le devis">
    Ouvrez l’action de soumission du devis.

    Renseignez les informations demandées. Joignez le document du devis si l’interface le propose.

    Relisez le montant et les notes avant d’envoyer.
  </Step>
</Steps>

## Résultat attendu

Votre réponse est transmise au gestionnaire.

Si vous avez envoyé un devis, le gestionnaire peut l’examiner. Il peut ensuite accepter votre proposition, demander des précisions ou choisir une autre suite pour l’intervention.

## Points d’attention

- Répondez seulement si vous avez assez d’informations pour vous positionner.
- Indiquez clairement ce qui est inclus dans le devis.
- Ajoutez une note si le montant dépend d’une visite ou d’un diagnostic.
- Ne soumettez pas un devis tant que les informations importantes ne sont pas vérifiées.
- Certaines actions peuvent dépendre du statut actuel de la demande.
