## Objectif

Cette page explique comment suivre une intervention acceptée en tant que prestataire.

Elle couvre les étapes principales : consulter l’intervention, gérer le créneau, intervenir sur place, documenter les travaux et indiquer que l’intervention est terminée.

## Quand utiliser cette page ?

Utilisez cette page lorsqu’une intervention vous est confiée après validation par le gestionnaire.

Elle est utile pour :

- retrouver une intervention acceptée ;
- consulter le créneau prévu ;
- proposer ou confirmer un créneau si nécessaire ;
- suivre les consignes avant le déplacement ;
- documenter les travaux ;
- marquer l’intervention comme terminée.

## Avant de commencer

Avant d’aller sur place, vérifiez :

- l’adresse et les informations d’accès ;
- le problème signalé ;
- le type d’intervention attendu ;
- le créneau prévu ou à proposer ;
- les notes du gestionnaire ;
- les documents utiles, comme le mandat, si disponibles.

Préparez aussi les photos ou notes que vous devrez ajouter après l’intervention, si l’interface le permet.

## Étapes

<Steps>
  <Step title="Ouvrir l’intervention acceptée">
    Depuis votre espace prestataire, ouvrez la liste des interventions à réaliser.

    Sélectionnez l’intervention concernée pour afficher son détail.
  </Step>

  <Step title="Vérifier les informations de l’intervention">
    Relisez le problème signalé, le lieu, la priorité et les informations utiles.

    Si un mandat ou un document est disponible, téléchargez-le avant le déplacement.
  </Step>
</Steps>

<Steps>
  <Step title="Gérer le créneau">
    Vérifiez si un créneau est déjà proposé ou confirmé.

    Si l’interface le permet, proposez un créneau au locataire ou confirmez le créneau proposé par l’autre partie.

    Ne confirmez un créneau que si vous pouvez l’honorer.
  </Step>

  <Step title="Démarrer l’intervention">
    Le jour prévu, suivez les consignes affichées dans l’intervention.

    Si l’action est disponible, utilisez le bouton de démarrage au début de la visite ou des travaux.
  </Step>

  <Step title="Documenter les travaux">
    Ajoutez les informations utiles pendant ou après l’intervention, si l’interface le propose.

    Indiquez ce qui a été constaté, ce qui a été fait, et les suites éventuelles.

    Ajoutez des photos si elles aident à comprendre le résultat.
  </Step>

  <Step title="Terminer l’intervention">
    Lorsque les travaux sont terminés, utilisez l’action prévue pour les marquer comme terminés.

    Si une nouvelle visite est nécessaire, suivez plutôt l’action indiquée par l’interface pour terminer la session ou proposer un nouveau créneau.
  </Step>
</Steps>

## Résultat attendu

L’intervention passe à l’étape suivante.

Après votre action de fin, le locataire ou le gestionnaire peut être amené à confirmer la clôture. Une fois validée, l’intervention est considérée comme terminée.

## Points d’attention

- Vérifiez toujours le créneau avant de vous déplacer.
- Ne démarrez l’intervention que lorsque vous êtes prêt à intervenir.
- Documentez les travaux de façon courte et factuelle.
- Ajoutez des photos si l’interface le permet et si elles sont utiles.
- Si les travaux ne peuvent pas être terminés en une fois, utilisez l’action adaptée plutôt que de marquer l’intervention comme terminée.
- Certaines actions peuvent apparaître ou disparaître selon le statut de l’intervention.
