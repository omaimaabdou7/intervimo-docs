## Objectif

Cette page explique comment suivre une intervention acceptée en tant que prestataire.

Elle couvre les étapes principales : consulter l’intervention, proposer un créneau, préparer le déplacement, suivre l’exécution et terminer les travaux lorsque l’action est disponible.

## Quand utiliser cette page ?

Utilisez cette page lorsqu’une intervention vous est confiée après validation par le gestionnaire.

Elle est utile pour :

- retrouver une intervention acceptée ;
- consulter les informations de l’intervention ;
- télécharger le mandat si nécessaire ;
- proposer un créneau ;
- suivre les consignes avant le déplacement ;
- documenter les travaux ;
- terminer l’intervention lorsque l’interface le permet.

## Avant de commencer

Avant d’aller sur place, vérifiez :

- l’adresse et les informations d’accès ;
- le problème signalé ;
- le type d’intervention attendu ;
- le niveau d’urgence ;
- le devis validé ;
- les documents utiles, comme le mandat, si disponibles.

Préparez aussi les notes que vous devrez ajouter après l’intervention si l’interface le permet.

## Étapes

<Steps>
  <Step title="Ouvrir l’intervention acceptée">
    Depuis votre espace prestataire, ouvrez la liste des interventions.

    Sélectionnez l’intervention concernée pour afficher son détail.
  </Step>

  <Step title="Vérifier les informations de l’intervention">
    Relisez le problème signalé, le lieu, la priorité et les informations utiles.

    Si un mandat est disponible, téléchargez-le avant le déplacement.
  </Step>

  <Step title="Gérer le créneau">
    Dans l’écran d’exécution, sélectionnez un jour dans le calendrier pour proposer un créneau.

    Vérifiez la date et l’heure avant d’envoyer votre proposition.

    <img
      src="/images/prestataire-execution-intervention.png"
      alt="Intervention en phase d’exécution avec le calendrier de proposition de créneau"
    />
  </Step>

  <Step title="Préparer l’intervention">
    Avant le rendez-vous, vérifiez que vous disposez des informations nécessaires pour intervenir.

    En cas d’information manquante, contactez le gestionnaire par le canal prévu dans l’application.
  </Step>

  <Step title="Réaliser les travaux">
    Le jour prévu, réalisez l’intervention selon les informations disponibles.

    Notez les éléments importants : diagnostic, action réalisée, pièce remplacée, ou suite à prévoir.
  </Step>

  <Step title="Terminer l’intervention">
    Lorsque les travaux sont terminés, utilisez l’action de fin si elle est disponible dans l’interface.

    Si une nouvelle visite est nécessaire, suivez l’action proposée par l’application plutôt que de marquer l’intervention comme terminée.
  </Step>
</Steps>

## Résultat attendu

L’intervention avance dans la phase d’exécution.

Le gestionnaire et le locataire peuvent suivre l’avancement. Une fois les travaux finalisés et validés, l’intervention peut être clôturée.

## Points d’attention

- Vérifiez toujours le créneau avant de vous déplacer.
- Ne proposez un créneau que si vous pouvez l’honorer.
- Téléchargez le mandat si vous en avez besoin sur place.
- Documentez les travaux de façon courte et factuelle.
- Si les travaux ne peuvent pas être terminés en une fois, indiquez clairement la suite à prévoir.
- Certaines actions peuvent apparaître ou disparaître selon le statut de l’intervention.
