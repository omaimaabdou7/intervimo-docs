## Objectif

Cette page explique comment suivre une intervention depuis l'espace locataire.

Elle vous aide à comprendre l'état de votre demande, à répondre au gestionnaire, et à agir lorsque l'interface vous le propose.

## Quand utiliser cette page ?

Utilisez cette page après avoir envoyé un signalement.

Elle est utile pour :

- vérifier l'avancement de l'intervention ;
- comprendre ce que signifie le statut affiché ;
- répondre à une demande de précision ;
- confirmer un créneau si un rendez-vous est proposé ;
- suivre les travaux jusqu'à leur fin ;
- valider ou confirmer les travaux si l'interface le permet.

## Avant de commencer

Ouvrez votre espace locataire depuis votre lien d'accès.

Depuis la page d'accueil, ouvrez l'intervention que vous souhaitez suivre. Vous pouvez consulter son statut, sa chronologie et, si disponible, la conversation avec le gestionnaire.

## Étapes

<Steps>
  <Step title="Ouvrir l'intervention">
    Depuis votre espace locataire, sélectionnez l'intervention concernée.

    La carte de l'intervention affiche déjà les informations principales : titre, description courte, niveau d'urgence et statut actuel.

    <img
      src="/images/locataire-suivre-liste-execution.png"
      alt="Liste des interventions dans l’espace locataire avec leur statut"
    />
  </Step>

  <Step title="Lire le statut">
    Consultez le statut affiché sur la carte ou dans le détail de l'intervention.

    Les statuts peuvent être présentés avec des mots simples :

    - Qualification : le gestionnaire examine le signalement.
    - Mise en relation : un prestataire est recherché ou sélectionné.
    - Exécution : l'intervention avance vers le rendez-vous, les travaux ou leur clôture.
    - Rejetée : la demande n'a pas été acceptée.
    - Annulée : l'intervention a été annulée.
    - Archivée : l'intervention est terminée et conservée dans l'historique.

    <img
      src="/images/locataire-suivre-statut.png"
      alt="Intervention affichée avec son statut dans l’espace locataire"
    />
  </Step>

  <Step title="Consulter la chronologie">
    Lisez les événements affichés dans la chronologie.

    Elle permet de voir les grandes étapes : signalement, qualification, prestataire assigné, travaux terminés.
  </Step>

  <Step title="Répondre à une demande de précision">
    Si le gestionnaire a besoin d'informations, ouvrez la conversation de l'intervention.

    Répondez simplement à la demande de précision. Ajoutez les détails utiles pour aider le gestionnaire à qualifier la demande.

    <img
      src="/images/locataire-suivre-demande-precision.png"
      alt="Demande de précision du gestionnaire dans la conversation de l’intervention"
    />
  </Step>

  <Step title="Confirmer un créneau si nécessaire">
    Si un créneau est proposé, vérifiez la date et l'heure.

    Confirmez le créneau si vous êtes disponible. Si l'interface le permet, proposez un autre créneau lorsque celui-ci ne vous convient pas.
  </Step>

  <Step title="Suivre les travaux">
    Lorsque l'intervention est en exécution, suivez les indications affichées.

    Certaines actions apparaissent seulement au bon moment, par exemple avant un rendez-vous, pendant les travaux, ou après leur fin.
  </Step>

  <Step title="Valider les travaux si l'interface le propose">
    Après les travaux, l'interface peut vous demander de confirmer que tout est terminé.

    Vérifiez le résultat avant de valider. Si quelque chose reste à traiter, suivez les options affichées dans l'application.
  </Step>
</Steps>

## Résultat attendu

Vous savez où en est l'intervention et quelle action est attendue de votre part.

Si aucune action n'est affichée, cela signifie généralement que le gestionnaire ou le prestataire doit poursuivre l'étape en cours.

## Points d'attention

- Les actions disponibles dépendent de l'étape actuelle de l'intervention.
- Une demande de précision doit être traitée rapidement pour éviter de ralentir la qualification.
- Ne confirmez un créneau que si vous êtes disponible.
- Avant de valider les travaux, vérifiez que le problème signalé est bien résolu.
- Si une information semble incorrecte, contactez le gestionnaire depuis la conversation lorsque celle-ci est disponible.

<Info>
  Certaines étapes peuvent être visibles sans action de votre part. Dans ce cas, vous pouvez simplement suivre l'avancement.
</Info>
