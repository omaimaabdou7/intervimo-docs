# Documentation Intervimo

<Badge color="#FD593E">Beta</Badge>

Intervimo aide les équipes immobilières à piloter les interventions, du signalement d'un problème jusqu'à la réalisation des travaux.

Cette documentation vous guide selon votre rôle dans l'application. Choisissez votre espace pour retrouver les actions à effectuer, les étapes à suivre et les points à vérifier.

<CardGroup cols={3}>
  <Card title="Locataires" icon="home" href="/locataires/signaler">
    Signalez un problème, ajoutez les informations utiles et suivez l'avancement de votre intervention.
  </Card>
  <Card title="Gestionnaires" icon="clipboard-list" href="/gestionnaires/qualifier">
    Qualifiez les demandes, sélectionnez les prestataires, gérez les devis et planifiez les interventions.
  </Card>
  <Card title="Prestataires" icon="wrench" href="/prestataires/recevoir-chiffrer">
    Recevez un mandat, chiffrez l'intervention, proposez un créneau et suivez l'exécution des travaux.
  </Card>
</CardGroup>
